LOTRO visual targets
--------------------
  Moon-white / pale blue rune lines (Moria west-gate feel)
  Dark cool stone — runes only visible as luminous script
  Ithildin ingot: starlight silver, softer than mithril blue
  Glowing swords: stronger ethereal bloom on elven script

Run upgrade
-----------
  python tools/upgrade_ithildin_runes.py
  python tools/upgrade_ithildin_runes.py --blocks-only
  python tools/upgrade_ithildin_runes.py --weapons-only

Note: Dwarven ithildin doors need textures from the Legacy mod JAR
(blocks/door*.png). Extract with:
  python tools/extract_mod_textures.py mod.jar --filter ithildin
